### 2. Insira seu arquivo html ja salvo no repositório, para isso use o terminal💕
* git init
* git add .
* git commit -m
* git remote + push (link do repositório) - copiar comandos diretamente em seu repositório


 ---

### 3. Criar GitHub Pages💕

* Vá em **Settings, Pages**
* Em **"Source"**, escolha:

  * Branch: `main`
  * Pasta: `/ (root)`
*salve pós processo
